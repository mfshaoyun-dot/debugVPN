## 关键发现

### 1. 基础套餐 OpenAI 可用，高级/旗舰不可用
- **基础套餐**的台湾/新加坡/日本/韩国/马来西亚节点：Claude ✅ + OpenAI ✅
- **高级/旗舰套餐**相同地区：Claude ✅ + OpenAI ❌（全部 000 超时）
- 原因：两者使用不同 IP（如基础新加坡用 .82/.85/.88/.91，高级用 .83/.86/.89/.92），OpenAI 对后者 IP 段有封锁

### 2. 香港节点质量最差
- HKT3（218.252.114.192）：Claude 返回 **403**，直接封锁
- HKT3-SS（旗舰/高级/基础）：反复 TIMEOUT
- 其余香港节点：Claude OK 但 OpenAI 全部 403 或 000

### 3. Gemini 全线不通
- 所有 110 个节点测试结果均为 000（超时）
- 建议：Gemini 可能需要 DNS 分流或特定出口区域

### 4. 法国节点高风险
- IP `51.178.79.136`：proxy=Y, hosting=Y → 风险 HIGH
- 是数据中心 IP，最容易被 AI 服务识别封锁
